PostgreSQL

PostgreSQL es una de las principales opciones para Bases de Datos Relacionales que requieran estructuras lógicas complejas. Junto con MySQL y MariaDB, compone el ecosistema de bases relacionales libres. La comunidad de PostgreSQL ha sabido llevar a este motor, hacia un framework de datos.
